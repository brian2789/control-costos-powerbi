# Dashboard de Control Presupuestario y Desvíos de Costos

Este proyecto consiste en un tablero interactivo desarrollado en **Power BI** conectado a una base de datos operativa en **Excel**, diseñado para el monitoreo financiero de proyectos de construcción y obras logísticas.

## 📊 Propósito del Dashboard y KPIs elegidos
El objetivo principal es transformar datos transaccionales en información estratégica para la toma de decisiones, permitiendo a la dirección identificar ineficiencias de forma inmediata.

* **KPI de Desvío Absoluto:** Calculado mediante lenguaje DAX (`Costo Real - Costo Presupuestado`), permite visualizar rápidamente qué proyectos están excediendo sus límites financieros y cuáles generan ahorros.
* **Segmentación por Rubro:** Filtro interactivo que desglosa el impacto del gasto en categorías clave (Materiales, Mano de Obra, Logística y Subcontratistas) para localizar el origen exacto de cada variación.

## 🔍 Principales Hallazgos Analíticos (Insights)
A partir del análisis consolidado de los datos, se detectaron los siguientes puntos críticos:
1. **Desvío Crítico en Infraestructura:** La *Obra Torre Av. Mitre* presenta el mayor desvío negativo del portafolio, superando los $200,000 de costo real por encima de lo planeado. El análisis por rubro demuestra que el desvío está fuertemente impulsado por el aumento en la categoría de **Materiales**.
2. **Eficiencia en Compras:** El proyecto *Ampliación Bodega* logró optimizar sus recursos en el rubro de Materiales, registrando un ahorro neto (desvío positivo) de $50,000 en comparación con el presupuesto inicial.
